task 3.1 (вес задания: 5)

В датасет norwegian_lexical_development.csv записаны данные из исследования [Hansen 2017], посвященной усваиванию лексики носителями норвежского. В датасете собраны следующие переменные:

  • Word_NW — норвежские слова;
  • Translation — перевод норвежских слова на английский;
  • CDS_freq — усредненная частота слова в речи, адресованной детям;
  • AoA — усредненный возраст освоения слова (в месяцах);
  • Lex_cat — лексические категории приписанная исследователями.

Постройте и визуализируйте регрессию, которая предсказывает возраст усвоения слова категории action words в зависимости от частотности в речи адресованой детям (я использовал логорифм по основанию 10 от этой переменной). Я использовал мягкое арпиорное распределение для сигмы (prior(normal(2, 5)), и достаточно строгие априорные распределения для случайного члена и углового коэффициента. Попробуйте их определить, опираясь на данные и графики ниже:

# mean_AoA = 24.8  -> центр prior на интерсепт
# sd_AoA = 2.57 -> sd prior на интерсепт
# sd_log = 0.648 -> sd prior на slope (≈ 0.7)
# correlation = -0.505
# sd_AoA   = 2.57 -> prior на slope
# -2 я пыталась как-то посчитать используя данные выше
# -2 ≈  -0.505 * (2.57 / 0.648)
priors_1 <- c(
   prior(normal(24.8, 2.57), class = "Intercept"),  # mean_AoA +- sd_AoA
   prior(normal(-2, 0.7),    class = "b"),           
   prior(normal(2, 5),       class = "sigma", lb = 0) # из условия задания
)

fit_action <- brm(
   AoA ~ CDS_freq_log10,
   data    = df_action,
   family  = gaussian(),
   prior   = priors_1,
   cores   = 4,
   iter    = 2000,
   warmup  = 1000,
   chains  = 4,
   control = list(adapt_delta = 0.85),
   silent  = TRUE
)

task 3.2 (вес задания: 5)

В датасет karel_speech_tempo.csv записаны данные из билингвального корпуса карельского языка. В датасете собраны следующие переменные:

  • speaker — код носителя;
  • n_vowels — количество гласных в высказывании;
  • duration — длительность высказывания в секундах;
  • gender — пол носителя;
  • speech_tempo — скорость речи в высказывании.

Постройте и визуализируйте регрессию со смешанными эффектами, которая предсказывает скорость речи на основании пола носителя. В качестве априорного распределения я использовал данные из работы [Stepanova 2011], согласно которой средняя скорость мужской речи 5.46 слогов в секунду со стандартным отклонением 2.02, а скорость женской речи 5.23 слогов в секунду со стандартным отклонением 1.84.

   prior(normal(2, 5),       class = "sigma", lb = 0),
   prior(normal(2, 10),      class = "sd")
)


fit_karel <- brm(
   speech_tempo ~ -1 + gender + (1|speaker), 
   data    = df,
   family  = gaussian(),
   prior   = priors,
   cores   = 4,
   iter    = 2000,
   warmup  = 1000,
   chains  = 4,
   control = list(adapt_delta = 0.85),
   silent  = TRUE
)
